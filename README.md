# S1.02 SAE - Octoverso

Projet de la SAÉ 1.02 (BUT Informatique) développé en C sous Visual Studio, en collaboration avec Ismail Fourra.

## Lancer le projet
- Ouvrir le fichier `SAE.sln` avec Visual Studio
- Compiler puis exécuter le projet (Ctrl + F5)

## Explication du projet
Octoverso est un jeu de société de plus de 15, les régles (disponibles en visionnant la vidéo youtube `https://www.youtube.com/watch?v=A4F2e_z1U0c)` sont assez complexes. 
Le but de ce projet était donc de récréer aussi fidèlement que possible ce "célèbre" jeu de société, en langage C.
Cela impliquait donc de la gestion de mémoire, de l'allocation dynamique `malloc`, de la gestion de tableau, de librairies et de tout plein d'aspect du langage C.
